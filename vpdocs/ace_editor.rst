Editor Shortcuts for Web VPython
================================

WebVPython uses a version of the ACE editor. 

..	list-table:: Web VPython editor shortcuts
	:widths: 50 50 50
	:header-rows: 1

	* - Action
	  - Windows/Linux
	  - Mac
	* - Find
	  - Ctrl-F
	  - Cmd-F
	* - Find next
	  - Ctrl-K
	  - Cmd-G
	* - Find previous
	  - Ctrl-Shift-K
	  - Cmd-Shift-G
	* - Replace 
	  - Ctrl-R
	  - Cmd-Option-F
	* - Replace all
	  - Ctrl-Shift-R
	  - Cmd-Shift-Option-F
	* - Select all
	  - Ctrl-A
	  - Cmd-A
	* - Toggle comment
	  - Ctrl-/
	  - Cmd-/
	* - Undo
	  - Ctrl-Z
	  - Cmd-Z
	* - Line end
	  - Alt-Right
	  - Cmd-Right
	* - Line start
	  - Alt-Left
	  - Cmd-Left
